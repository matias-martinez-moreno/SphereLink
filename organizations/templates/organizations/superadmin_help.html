Quick Actions
Create Organization
  1. Click "Create Organization" button
  2. Fill name, email, description, phone
  3. Set status (Active/Inactive)
  4. Click "Create Organization"
Add Users
  1. Go to organization → "Add New User"
  2. Fill username, email, password
  3. Set role: Staff or Member
  4. User is immediately added to organization
Organization Management
View Organizations
  • See all organizations in dashboard
  • Search by name, email, description
  • View user count and statistics
Edit Organization
  • Click "Edit Organization Info"
  • Update name, email, description
  • Change active/inactive status
Manage Users
  • Add new users to organization
  • Change roles (Staff/Member)
  • Reset passwords or delete users
User Management
Manage User Roles
  • Go to organization → "Manage Organization Users"
  • Click "Edit Role" to change Staff/Member
  • Click "Change Password" to reset user password
  • Click "Delete User" to remove from organization
Bulk Invite Users
  • Click "Bulk Invite Users" in organization
  • Upload CSV file with user data
  • Users are automatically created and invited
User Roles Explained
Staff
  • Create Official events
  • Moderate content
  • Delete any event/comment
Member
  • Create Community/Org events
  • Join events and comment
  • Cannot create Official events
Super Admin
  • Full system control
  • Manage all organizations
  • Control all users
Contact Messages
View Messages
  • See all user contact messages
  • Filter by status (Pending/Solved)
  • Mark as solved when resolved
Statistics
  • View pending vs solved count
  • Search message content
  • Track response times
Pro Tips
Efficiency Tips
  • Use search to quickly find organizations/users
  • Use CSV bulk invite for multiple users
  • Monitor organization statistics regularly
Important Notes
  • Always verify before deleting organizations
  • Reset passwords when users can't login
  • Respond to contact messages promptly
