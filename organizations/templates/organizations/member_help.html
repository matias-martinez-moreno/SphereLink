Common Tasks
Creating Events
  1. Click "Create Event" in sidebar
  2. Fill in event details (title, description, date, location)
  3. Choose event type (Community or Organization)
  4. Set capacity and other options
  5. Click "Create Event"
Managing Events
  1. Go to "My Events" to see all your events
  2. Click on an event to view details
  3. Edit event details or manage registrations
  4. Delete only your own events
  5. Note: You can only manage events you created
Event Types You Can Create
Community Events
  • Created by members
  • Community-driven
  • You can create these!
Organization Events
  • Internal events
  • Organization members only
  • You can create these!
Troubleshooting
Event Creation Issues
  • Missing Fields: Make sure all required fields are filled
  • Past Dates: Events cannot be created for past dates
Access Issues
  • Cannot Create Official Events: This is normal - only staff can create them
  • Organization Access: Contact admin if you can't access organization features
